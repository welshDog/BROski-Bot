# Contributing to BROski-Bot

First off — thank you! 🧠 BROski is an ADHD-first Discord bot and we welcome all contributors.

## 🌟 How to Contribute

1. **Fork** the repo
2. **Create a branch** — `git checkout -b feature/your-idea`
3. **Make your changes** — keep commits small and focused
4. **Push** — `git push origin feature/your-idea`
5. **Open a PR** — describe what you changed and why

## ⚠️ Rules

- Never commit `.env` files or `DISCORD_BOT_TOKEN`
- Run `git rm -r --cached __pycache__` if you accidentally stage cache files
- Always `git fetch` before pushing

## 💬 Questions?

Open a [Discussion](https://github.com/welshDog/BROski-Bot/discussions) or join our Discord community!
